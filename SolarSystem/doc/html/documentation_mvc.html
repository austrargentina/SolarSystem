class MVC.StartScreen.ComputeEventsStart¶

Ist die Controller-Klasse des StartScreens

__init__()¶

Konstruktor

Erstellt unser Fenster und füllt es mit vorgegebnen Content; Überprüft auf Events und ruft dementsprechende Methoden auf.

Verarbeitet Tastertur-Knopf Events
